# GetZoneControlAngle
Used to calculate the angle for the *ZoneControl_Fill* image.  

Target is *SpectatorHUD* or *SpectatorHUD_Random*.  

## Node

## Inputs
**None**

## Outputs
|Name       |Type   |Description                |
|-----------|-------|---------------------------|
|**Angle**  |*Float*|An angle from -90.0 to 90.0|
