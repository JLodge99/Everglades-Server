# SpeedToUnits
Converts from target's speed to Unreal Units so the target can move in proportion to
the world.  

Target is *EvergladesMacroLibrary*.  

## Node

## Inputs
|Name       |Type   |Description                |
|-----------|-------|---------------------------|
|**Speed**  |*Float*|The target's speed value.  |

## Outputs
|Name       |Type   |Description    |
|-----------|-------|---------------|
|**Units**  |*Float*|Unreal units.  |
