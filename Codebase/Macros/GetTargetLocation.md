# GetTargetLocation
Gets a world location for a unit by adding the *TargetLocation* relative position to the
parent group's world position.  

Target is *SwarmUnit*.  

## Node

## Inputs
**None**

## Outputs
|Name           |Type       |Description                        |
|---------------|-----------|-----------------------------------|
|**ReturnValue**|*Vector*   |The unit's position in the world.  |
