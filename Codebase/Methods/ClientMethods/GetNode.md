# GetNode
Given a node's ID, returns the node object.  

Target is *EvergladesGameMode*.  

## Node

## Inputs
|Name   |Type   |Description    |
|-------|-------|---------------|
|**In** |*Exec* |Execution pin. |
|**NodeID**|*Integer*|The ID of the desired node.|

## Outputs
|Name       |Type       |Description        |
|-----------|-----------|-------------------|
|**Node**   |*NodePoint*|The target node.   |
