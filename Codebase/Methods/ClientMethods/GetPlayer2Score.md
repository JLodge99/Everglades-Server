# GetPlayer2Score
Bound to *Score_Red*, it returns Player 2's score as text.  

Target is *SpectatorHUD* or *SpectatorHUD_Random*.  

## Node

## Inputs
**None**

## Outputs
|Name           |Type   |Description        |
|---------------|-------|-------------------|
|**ReturnValue**|*Text* |Player 2's score.  |
