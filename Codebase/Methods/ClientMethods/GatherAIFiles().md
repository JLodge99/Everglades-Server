# GatherAIFiles()
This creates a list of the Everglades AI agents file names found in the *agents* folder.  

## Syntax
```cpp
GatherAIFiles(FString serverBasePath)
```

## Returns
An array of the AI agent file names.  

## Parameters
|Parameter          |Description                                        |
|-------------------|---------------------------------------------------|
|**serverBasePath** |The absolute location of the Everglades server.    |
