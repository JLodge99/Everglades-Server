# Generate3DNodeID()
Calculates the node ID given the x, y, and z

## Syntax
```cpp
Generate3DNodeID(int z, int y, int x, int zLen, int yLen, int xLen)
```

## Returns
An integer 

## Parameters
|Parameter                  |Description                                                                            |
|---------------------------|---------------------------------------------------------------------------------------|
|**z**                      |z index                                                                                |
|**y**                      |y index                                                                                |
|**x**                      |x index                                                                                |
|**zLen**                   |Length of z axis                                                                       |
|**yLen**                   |Length of y axis                                                                       |
|**xLen**                   |Length of x axis                                                                       |
