# GetUnit
Given an index of the *GroupUnits* array, returns the desired unit.  

Target is *SwarmUnit*.  

## Node

## Inputs
|Name   |Type       |Description                    |
|-------|-----------|-------------------------------|
|**ID** |*Integer*  |*Units* index of given unit.   |

## Outputs
|Name       |Type       |Description                |
|-----------|-----------|---------------------------|
|**Valid**  |*Boolean*  |Whether the unit was found.|
|**Unit**   |*SwarmUnit*|The desired unit.          |
