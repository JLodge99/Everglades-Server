# SetUnitHealth
Gets the unit and calls *SetNewHP* event from *SwarmUnit*.  

Target is *SwarmGroup*.  

## Node

## Inputs
|Name       |Type       |Description                |
|-----------|-----------|---------------------------|
|**Health** |*Float*    |The health of the unit.    |
|**UnitID** |*Integer*  |The ID of the unit.        |

## Outputs
**None**
