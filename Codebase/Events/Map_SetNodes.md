# Map_SetNodes
Called from *RunEvents* event in *EvergladesGameMode*, it initializes nodes in the
map and minimap.  

Target is *SpectatorHUD* or *SpectatorHUD_Random*.  

## Node

## Inputs
|Name       |Type                   |Description            |
|-----------|-----------------------|-----------------------|
|**Nodes**  |*Array\<NodePoint\>*   |The nodes in the match.|

## Outputs
**None**
