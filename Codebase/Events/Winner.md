# Winner
Called by *EvergladesGameMode* when there is a winner, it makes the winning 
message visible.  

Target is *SpectatorHUD* or *SpectatorHUD_Random*.  

## Node

## Inputs
|Name       |Type       |Description            |
|-----------|-----------|-----------------------|
|**Player** |*Integer*  |The ID of the player.  |

## Outputs
**None**
