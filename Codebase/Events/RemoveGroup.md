# RemoveGroup
Bound to the *GroupLeft* dispatcher, it removes the group from the *OccupyingGroups*
array and sets the group's *InNode* variable to **Null**.  

Target is *NodePoint*.  

## Node

## Inputs
|Name       |Type           |Description                        |
|-----------|---------------|-----------------------------------|
|**Group**  |*SwarmGroup*   |The group that has left the node.  |

## Outputs
**None**
