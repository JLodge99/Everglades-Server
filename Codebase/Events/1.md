# 1

Target is *EvergladesSpectatorPawn*.  

## Node

## Inputs
**None**

## Outputs
|Name           |Type           |Description                        |
|---------------|---------------|-----------------------------------|
|**Pressed**    |*Exec*         |Execution path if key was pressed. |
|**Released**   |*Exec*         |Execution path if key was released.|
|**Key**        |*Key Structure*|A reference to the key.            |
