# ToggleSwarmVisibility
Toggles between swarms and cube depending on the value of *SimplifiedMap* variable from
*EvergladesGameMode*.  

Target is *SwarmGroup*.  

## Node

## Inputs
|Name       |Type       |Description                        |
|-----------|-----------|-----------------------------------|
|**Visible**|*Boolean*  |Whether the unit should be visible.|

**None**
