# AddGroup
Bound to *GroupEntered* dispatcher, it adds the group to the *OccupyingGroups*
array and sets the group's *InNode* variable.  

Target is *NodePoint*.  

## Node

## Inputs
|Name       |Type           |Description                            |
|-----------|---------------|---------------------------------------|
|**Group**  |*SwarmGroup*   |The group that has entered the node.   |

## Outputs
**None**
