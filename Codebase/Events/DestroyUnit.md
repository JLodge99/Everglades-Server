# DestroyUnit
Called from *SetNewHP* event in *SwarmUnit*, calls functions from other blueprints to remove
the unit from play.  

Target is *SwarmGroup*.  

## Node

## Inputs
|Name       |Type       |Description            |
|-----------|-----------|-----------------------|
|**Unit**   |*SwarmUnit*|The unit to destroy.   |

## Outputs
**None**
