# SetTeamColors
Sets the color of the drone (red or blue) according to the owning player's ID.  

Target is *HeavyDrone*, *LightDrone*, or *ScoutDrone*.  

## Node

## Inputs
|Name           |Type       |Description            |
|---------------|-----------|-----------------------|
|**PlayerID**   |*Integer*  |The ID of the player.  |

## Outputs
**None**
